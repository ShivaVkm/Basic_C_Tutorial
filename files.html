File Operations & Functions

File Pointers

It is not enough to just display the data on the screen. We need to save it because memory is volatile and its content would be lost once program terminated, so if we need some data again there are two ways one is retype via keyboard to assign it to particular variable, and other is regenerate it via programmatically both the options are tedious. At such time it becomes necessary to store the data in a manner that can be later retrieved and displayed either in part or in whole. This medium is usually a file on the disk.

Introduction to file
Until now we have been using the functions such as scanf, printf, getch, putch, etc. to read and write data on the variable abd arrays for storing data inside the programs. But this approach poses the following programs.
1. The data is lost when program terminated or variable goes out of scope.
2. Difficulty to use large volume of data.

We can overcome these problems by storing data on secondary devices such as Hard Disk. The data is stored on the devices using the concept of file.
A file is a collection of related records, a record is composed of several fiels and field is a group of character.

The most straight forward use of file is via a file pointer.
FILE *fp;
fp is a pointer to a file.
The type FILE, is not a basic type, instead it is defined in the header file stdio.h, this file must be included in your program.

File operations

1. Create a new file
2. Open an existing file
3. Read from file
4. Write to a file
5. Moving a specific location in a file (Seeking)
6. Closing file

Opening a file

fp=fopen(filename, mode);
The filename and mode both are strings.

Here modes can be
"r" read
"w" write,overwrite file if it exists
"a" write, but append instead of overwrite
"r+" read and writ, do not destroy life if it exists
"w+" read and write, but overwrite file if it exists
"a+" read and write, but append instead of overwrite
"b" may be appended to any of the above to force the file to be opened in binary mode rather than text mode.

Eg.
FILE *fp;
fp=fopen("input.txt","r"); //opens input.txt file in read mode
fclose(fp); //close file

Sequential file access is performed with the following library functions.
1. fopen() - create a new file
2. fclose() - close file
3. getc() - read character from file
4. putc() - write character to a file
5. getw() -read integer from file
6. putw() - write integer to a file
7. fprintf() - write set of data values
8. fscanf() - read set of data values
